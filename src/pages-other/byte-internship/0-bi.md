# 4.24 - 出发前一天

本专题用于记录我在字节跳动实习经历/北漂经历，算是个人日记吧，出于保密要求，我更多的会记录一些北漂的感受和经历，技术方面的内容会比较少。

今日是去北京的前一天，因此开设本专题，后续会持续更新~

## BG

### 技术栈

我是中南大学 23 级的学生，主要学习 cpp，golang 相关技术栈，这里先说一下我的技术栈，截至 2026 年 4 月：

- C++: 语言层面可以熟练使用 11~23 的特性，相关的库杂七杂八非常多，毕竟各个东西不能重复造轮子对吧，包括配套的工具链，比如 CMake/Ninja 一类的构建工具

- Golang: 这个学的比较浅，只是大地老师的基础知识 + 极客时间的一个完整课程，然后在看 IAM 的一个项目，还没看完，但获益良多

- 网络: 因为做的是后端开发，这方面还是比较熟悉的，TCP/IP、HTTP/HTTPS、WebSocket、RPC 等协议都比较熟悉，RESTFul API 和 gRPC 的服务都能比较熟练的搭建，相关的库也能熟练使用，单机做过 7w+ QPS 的服务，分布式做过 100w+ QPS 的服务

- 并发: 主要是基于 Cpp 的并发编程，在我的学习笔记里有详细博客，做过很多个 **亿级** 吞吐的无锁结构，内存屏障和 MESI 等一类底层原理也是比较熟悉的，golang 的并发编程也有一些了解，不过仅限于调库，而没有深入到底层

- 数据库: 对 Mysql/MariaDB、Redis/ValKey、MongoDB 使用比较熟练，包括一系列八股，对原理还算清楚

- Linux: I use Arch, btw，所以对 Linux 的使用上比较熟练，但仅限于使用，我并未系统学习过 Linux 内核的知识，命令的调用也仅限于常用命令这样子

- 其他: 还有很多杂七杂八的技术栈，因为用的不多，但都做过一些小玩具，这里一起说了，比如 Docker、CI/CD、分布式系统、消息队列、LVS/KeepAlived、Nginx、Python、Vue、React、UE、SDL、EasyX 等等

技术栈这一块其实很难去说请我都会什么，很多技术不值得单独开一个分类，但它确实是我们必须的，组成我们能力的一个重要部分，且出于对技术感兴趣的目的，我总会去学一些对职业来说没什么用，但能取悦我的东西。

后面我也想学很多其他的技术栈，目前的话就先把 Golang 复习好吧，找实习的主语言是 C++，Golang 已经是只能看懂了，hhh，再看一下 IAM。

## 感想

说实话，看到最终录用邮件的时候真的想哭，任何一个在 2026 找过相关工作且不是运气好到一次过的同学应该都能理解，今年的难度和往年完全不是一个级别的，我的面试已经很顺利了，10/11 的通过率，但冗长的周期依旧让人崩溃，晚上焦虑的睡不着，早上一点消息声音就不敢睡，生怕错过任何一条消息导致自己失败，这一个月可以说是煎熬，同学们的笑声都加上了勉强的感觉，实在太难了。

但最终结果出来的一瞬间，一切都是值的，我们日常的学习，以及所有的努力并没有白费，它们终于被看到了，得到了真实的承认。

我很难说自己这一个月是快乐的，但它绝对是有意义的，每天和舍友中午开学，干到晚上 9 点，大家忙里偷闲在湖中凉亭谈谈理想，谈谈未来，上大学后第一次感受到了高中那种战友的感觉，我不后悔选择这条路，因为我是因为对技术的热爱才放弃了保研，我愿意用我接下来最合适的年龄继续在这个方向学习，成为我所向往的工程师。

这一个月差不多就是这样，大家累的同时也能乐呵呵的生活，就不再沉湎于此了，明天开始就是一个新的篇章，全是第一次，兴奋、紧张一类的情绪萦绕在我心头，明天的话就先把房子找一下，希望周日晚上前能签一个。

明天见~
