# 5.15 - 出发去合肥

现在是 5.16 日早上 10:40，补一下昨天的日记吧。

## 上午

昨天上午还在吃饭时，悉尼那边的同事回了我昨天发群里的报错日志，然后麻溜的吃完饭就跑去看看他反馈的问题，那个翻译功能让我以为他按我说的方法做了呢，结果他只是说让我换个机房部署，跟我 mt 教我的有些出入，然后还更碰巧的是 qa 那边在一个什么环境都没变化的情况下居然跟昨晚不一样，给的 logid 能拿到我这个服务的日志了，我还真以为 ok 了呢。

接着就是在 debug，流程和昨晚差不多，没啥值得特别说的，就是一步一步的搞。

但是在 debug 过程中，我这个细方向的组的 赵哥 跟对方沟通说我是新同学，上来就搞这种需求对新同学实在不太友好，然后听着意思是让我负责一个小一点的需求，然后这个大需求交给其他人负责，说实在的，这个慢慢调试肯定没毛病，但是很累，毕竟除了 qa 我一个都不认识，该找谁全得问 mt，人际上很不友好，等 mt 过来的时候说把 mr 发给他，他来继续。

这个就放一起说了，中间和 mt 说了一下我写代码的思路，然后他那边给出的反馈是没问题，还是我昨天 debug 上出的问题，不过也还好了，这下轻松了不少。

## 下午

因为这周六日要参加合肥的徽客松，来当一手小评委，所以提前一个小时下班了。

下午的话和组里一起去吃了个湖北菜，然后因为没有需求了，我正好把串讲的初版写了个七七八八，只剩后续一些整理目录方面的东西了，因此就有一下午的空闲时间。

我就在看各种文档，从 cpp 的 debug、调优、规范 看到 agent 的全流程，不得不吐槽一下，字节的 cpp 版本真的好低，gcc 还用的是 gcc4/8，岂不是证明最多写个 c++11，那我这种现代 cpp 选手直接砍断一条腿啊。

但是这边 golang 就挺新的，学到的东西都可以用的。

然后 5 点多打车来首都机场，这里必须抨击一下电车这种东西，搞了个傻逼的动能回收，配上师傅的风琴脚，能不晕车的我佩服好吧，果然还是油车舒服一些。

这是第一次登机，这里总结一下昨天的流程：

```
自助值机（选位置）
-> 机票上有说值机柜台，点两下就可以拿到登机票（这一步选择要不要托运行李）
-> 然后按照目的地分国内外，走进对应门户
-> 安检（摸到有点恶心，尤其是夏天穿的薄，怪不得都是女生在安检，要是男的不得飞起来）
-> 找登机口
-> 剩下就和高铁一模一样了，给他登记牌扫一下就好
```

坐飞机的体验的话感觉没啥特殊的，只是鼓膜有概率会突然难受一下，然后就是没有网络，下次可以提前缓存一些不用耳朵的东西在路上消遣。

剩下时间就全是赶路了，到酒店之后就是倒头就睡，真的力竭了。

昨天差不多就这样，期待今天的徽客松，看看现在的学生们的创意～
