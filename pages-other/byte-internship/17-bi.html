5.16 - 徽客松初体验

现在是 5.16 晚上 23:56，写一下今天的经历。

上午

今天上午的话没啥好说的，虽然徽客松上午开幕，但我感觉没啥必要，因此选择在酒店呆着。

这边早餐是自助，因此早上直接爽吃，猛干两大盘子，然后回房间补了一下昨天的日志，然后就是酷酷刷视频，中间就洗了个澡，剩下时间就很摆。

到了中午去吃老乡鸡，好吃，蒸鱼、白切鸡 都好吃，鸡汤不好喝，潇洒一波，一个人一顿饭 3 个菜，你敢想吗（bushi）。

下午

吃完饭就去打车去那边了，见到了其他三位字节的同学，因为他们才刚开始开发，所以我们就在这边等着，我是又一次看飞雪的 golang 基础，毕竟时间很碎片，看点碎片的也正常。

然后过了一会就去巡场了，看看他们做的产品，交流一下，如果有疑问的就解答一下，说实话有点尴尬的，毕竟他们这边基本全 vibe，大家没有讨论技术的必要，只有对产品和 ui 的需求，因此今天下午说实话挺憋屈，这也提醒我技术同学也是必须要学产品的，所谓的产品思维还是非常重要的。

中间忙里偷闲，又看了两篇文档，明天晚上回酒店了看看能不能把这几篇全搞完，要不然还是有点无聊了，而且那个 iam 也没法推，唉，感觉想学的好多，我的 im 系统有很多想做的优化、iam 也是很好的项目、以及想学 agent 相关的内容，但是经力又跟不上，还是好好规划一下吧，要不然感觉进步全依赖平台，古法和ai的对抗，真的很难取舍。

然后晚上的围炉会有意思，其实就是单纯的围在一起听听彼此吹牛逼，今晚见到了几个人，我觉得还是比较值得说一下：

经历感想
同学 A高二，参加多场黑客松，手里拿过 n 个省奖国奖，从初中开始就开始赚钱了我真说白了，高二就给我好好刷题啊，你这样学长们压力很大啊，konoyalu，唉，有点见识到现在的后生有多牛逼，有一种被拍死在沙滩的感觉。 ps，这个同学就是刻板印象那种成都圆脸络腮胡，太典型了，同学我不是人身攻击，看到了不要骂我，我只是感慨一下
同学 B港科大博士后，有公司，在做 agent network 这个方向这个真的，一开口那个声音，真的让人感知到什么叫高知，讲了一个白血病孩子的父母愿不愿意让他们蒸馏一下他们的孩子，以及很多其他的，说实话学到很多，但是更想感慨的是知识的气质，原来有东西的人真的只用一个措辞就有差距，同样是一个事情，有的人嘴里说出来就是很牛，就很可靠
同学 CPicoHeart 做 agent 开发这个纯纯sb，上海人，完全刻板印象，一句话不说两句英语会死啊，又不是技术场景非说不可，那个音色、喝咖啡的表情、说话的态度，没见识过的可以看看金兑演的，我真以为金兑是玩梗，真有这种傻逼，PicoHeart 是字节全资子企业，但你不至于光说自己是字节的吧，这么看不起自己公司吗，我拉你进评委群都拉不了，显示你是外部人员懂吗？我旁边几个正职都没你傲，弱智一个

但整体说来，围炉会是很有意思的，我们聊到 22:37 才散场，听清晨说明天的活动更有意思，下午一点开始，到时候直接再好好玩一下。

今天差不多就这样，明天见～
